# README - Menjalankan Project React Native / Expo

## Persiapan

Pastikan sudah menginstall:
1. Node.js** (LTS) → [https://nodejs.org](https://nodejs.org)
   Cek versi: `node -v` dan `npm -v`
2. Git** → [https://git-scm.com/downloads](https://git-scm.com/downloads)
   Cek versi: `git --version`
3. Expo CLI
   Install: `npm install -g expo-cli`
   Cek versi: `expo --version`
4. Install aplikasi Expo Go di HP (bisa diunduh di Play Store atau App Store)

## Clone Repository dari GitHub

Cari URL repo GitHub project, lalu jalankan:
1. git clone https://github.com/almaalfiatul/react-native-project
2. cd repository

## Install Dependencies
npm install

## Menjalankan Aplikasi
npm start
 atau
expo start

## Scan QR code dengan **Expo Go** di HP untuk langsung menjalankan. 

## React Native CLI:
* Android:
npx react-native run-android

* iOS (hanya di Mac):
npx react-native run-ios


## Troubleshooting

* Pastikan Node.js, npm, dan Expo CLI versi kompatibel.
* Pastikan emulator sudah siap sebelum menjalankan simulasi.
* Jika ada error missing dependencies, ulangi `npm install` atau `yarn install`.
